# Finance Excel Workspace

## 절대 규칙
- 2H26_Outlook_Raw/ 폴더 원본 파일 절대 수정 금지 — 읽기 전용으로만 접근
- 숨김 시트(sheet_state = 'hidden') 완전 무시 — 존재하지 않는 것으로 취급
- 출력은 항상 2H26_Outlook_Output/ 에 새 파일로만 생성
- 불명확하면 작업 전에 반드시 먼저 물어볼 것

## 차트 영문 번역 작업 방식

### 1. 수집 기준
- 차트가 1개 이상 있는 시트만 대상
- 숨김 시트 완전 제외
- 차트 없는 시트 제외

### 1-1. 중복 차트 제거
- 차트 제목 + 데이터 범위 + 계열 이름이 모두 같으면 동일 차트
- 중복 차트는 1개만 번역, 나머지 무시
- 작업 전 중복 판단 목록 보여주고 확인 받을 것

### 2. 작업 방식
- 원본 파일을 shutil.copy로 복사 → Output 폴더에 저장
- 복사본에만 작업, 원본 파일 절대 수정 금지
- 원본 시트는 절대 건드리지 않음

### 3. 출력 구조
별도 시트 생성 없이 기존 시트 안에 나란히 배치:
- 원본 국문 차트는 현재 위치 그대로 유지 (절대 손대지 않음)
- 영문 차트를 국문 차트 오른쪽에 추가 배치
- 영문 차트 크기/타입은 국문 차트와 동일하게

### 4. 영문 차트 생성 방식

#### 범례/텍스트가 셀 참조인 경우 (=Sheet1!$B$1 형태)
- 복사된 셀의 한국어 텍스트를 영문으로 교체
- 영문 차트는 복사된 영문 셀을 참조하도록 새로 생성
- 글꼴 Arial Narrow 적용

#### 범례/텍스트가 차트 내부 직접 입력인 경우
- 차트 XML의 텍스트를 직접 영문으로 교체
- 글꼴 Arial Narrow 적용

### 5. 번역 절차
- 번역 전 한국어 텍스트 목록 추출해서 먼저 보여줄 것
- 번역 대상:
  - 차트 제목
  - 축 제목 (X축, Y축)
  - 범례 텍스트 (직접 입력 + 셀 참조 둘 다)
  - 단위 텍스트 (조원), (억원), (%YoY) 등
  - 차트에 표시되는 계열명
  - X축 카테고리값 중 한국어 (업종명, 기업명, 항목명 등)
- 번역 제외:
  - 숫자 데이터
  - 차트에 표시되지 않는 셀 데이터
- 확인 받고 나서 적용
- 번역 후 원본 텍스트와 1:1 대조해서 누락 검증

### 6. 파일 1개씩 진행
- 파일 1개 완성 후 나한테 확인 받고 다음 파일 진행
- 테스트 3회 이상 실패하면 중단하고 나한테 보고할 것

## 폴더 구조
```
2H26_Outlook_Raw/          ← 원본 (절대 수정 금지)
2H26_Outlook_Output/       ← 결과물
2H26_Outlook_Reference/    ← 예시 파일과 업종 및 종목 영문번역 버전
```
